주요 기능

간편 모드

좌우 수평계, 조정 방향, 예상 하늘 방향을 한 화면에 표시해 북쪽 기준을 빠르게 잡습니다.

상세 측정

RA축의 반복 자세와 센서 상태를 확인해 장착 상태와 조정 근거를 더 자세히 살핍니다.

음성·햅틱 안내

원할 때만 현재 조정 방향을 듣고 느낄 수 있어, 화면을 계속 보지 않아도 됩니다.

PolarPilot은 iPhone 센서 기반 안내 도구입니다. 실제 별 관측, 정밀 극축 정렬 완료, 촬영 성공을 보증하지 않습니다.

사용 방법

1

iPhone 올리기

경통 또는 브래킷에 iPhone을 안정적으로 올리고 좌우 수평을 먼저 맞춥니다.

2

모드 선택

빠른 안내는 간편 모드, 장착 상태 확인은 상세 모드를 선택합니다.

3

RA축 측정

가능한 범위에서 RA축을 다섯 정지 자세로 움직여 회전 일관성을 확인합니다.

4

마운트 조정

화면 안내를 보며 방위각과 고도 조정 나사를 조금씩 움직입니다.

iPhone용으로 개발 중입니다.

현재 실제 장비와 iPhone에서 검증을 진행하고 있습니다. 배포 준비가 되면 이 페이지에서 안내합니다.
